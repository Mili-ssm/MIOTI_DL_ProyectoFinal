# MIOTI_DL_ProyectoFinal

Proyecto Final del master de Mioti en DeepLearning.


## Instalar UV

- **Windows**

    ```bash
        powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```

- **MacOS / Linux**

    ```bash
        curl -LsSf https://astral.sh/uv/install.sh | sh
    ```

- **Pip**

    ```bash
        pip install uv
    ```

## Instalar Dependencias

```bash
    uv sync
```


## Ejecutar tests

```bash
    uv sync
    uv run pytest
```
